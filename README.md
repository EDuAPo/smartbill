## 📱 应用简介

SmartBill 财伴是一款基于 AI 的智能记账应用，通过手机号即可快速登录，随时随地记录和分析您的消费情况。

## ✨ 核心功能

### 1. 智能记账
- **AI 拍照识别** - 拍照即可自动识别账单金额、商户和分类
- **手动记账** - 快速记一笔，记录日常消费
- **相册导入** - 从相册选择图片批量识别账单

### 2. 数据分析
- **收支报表** - 清晰展示收入支出情况
- **分类统计** - 按餐饮、购物、交通等分类查看消费占比
- **预算管理** - 设置月度预算，实时监控消费进度

### 3. AI 智能助手
- **语音对话** - 向 AI 助手咨询财务问题
- **智能分析** - 获取个性化的消费建议和理财提示

### 4. 个性化设置
- **多主题模式** - 支持深色/浅色主题切换
- **自定义头像** - 多种头像可选
- **昵称修改** - 个性化您的账户名称

## 🔐 账户与数据

- **手机号登录** - 使用手机号接收验证码即可登录
- **数据永久保存** - 同一手机号登录，数据永不丢失
- **本地存储** - 数据存储在本地，保护隐私安全

## 🛠️ 技术栈

- **前端框架**: React + TypeScript
- **路由**: React Router
- **AI 能力**: 阿里云千问 (Qwen)
- **样式**: Tailwind CSS
- **部署**: Vercel

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```
